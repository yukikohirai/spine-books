# 本を1冊追加する方法

このサイトは **`data/books.json` に1ブロック追加するだけ** で、
一覧ページ・個別ページ・カテゴリ絞り込み・くるくるカルーセルの全部に自動で反映されます。
HTMLは触りません。

## 手順

1. `data/books.json` を開く
2. 一番最後の `}` のあとに `,` を付けて、下のテンプレートを貼り付けて埋める
3. 保存（公開後は GitHub で編集 → 自動でサイト更新）

## テンプレート

```json
{
  "slug": "eigo-no-mijikai-namae",
  "title": "本のタイトル",
  "author": "著者名",
  "cover": "images/xxx.jpg",
  "to": "〇〇な人へ",
  "genre": "小説",
  "description": "一覧カードに出る短い紹介（2〜3行）。",
  "comment": "個別ページにだけ出る長い感想。\n改行は \\n で書く。空でもOK。",
  "amazon_url": "https://www.amazon.co.jp/dp/XXXXXXXXXX",
  "rakuten_url": ""
}
```

## 各項目の意味

| 項目 | 説明 |
|---|---|
| `slug` | その本のURL（例: `book.html?id=convenience-store-woman`）。英小文字とハイフンのみ。**他の本と重複NG** |
| `title` / `author` | タイトルと著者 |
| `cover` | 表紙画像。`images/` フォルダに入れてファイル名を書くのが確実 |
| `to` | **「誰に渡したいか」カテゴリ**。同じ文言の本同士が自動でグループになるので、表記を揃える |
| `genre` | ジャンル（今は予備。将来の絞り込み用） |
| `description` | 一覧カード＋個別ページに出る短文 |
| `comment` | 個別ページ専用の長文感想。空（`""`）なら非表示 |
| `amazon_url` / `rakuten_url` | 購入リンク。アフィリエイトリンクをそのまま貼ってOK。無ければ `""` |

## 既存の「誰に渡したいか」カテゴリ

- 何かに夢中になったことがある人へ
- 「普通」に疲れた人へ
- 人間関係のパターンに気づきたい人へ
- 自分を責めがちな人へ
- 情報に流されたくない人へ

新しいカテゴリを作るのも自由（`to` に新しい文言を書くだけ）。
